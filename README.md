# Pflanzenlern-App final v4

Neue Funktionen:
- Farben einstellbar, inklusive Original-Pink (#ffa1c5) und Tree-Node-Farben
- Pflanzennamen werden mit Leerzeichen angezeigt, interne Originalnamen bleiben erhalten
- Notizen aus älteren Versionen werden automatisch übernommen, sofern sie im selben Browser/auf derselben GitHub-Pages-Domain gespeichert waren
- Eigene Bilder können einzeln gelöscht werden
- iNaturalist-Bilder sind anklickbar und öffnen groß
- Baum-Zoom ist stärker begrenzt plus Button „Baum zentrieren“
- Kartenansicht mit MyMaps/KML/GeoJSON-Import
- Plot-Klick setzt automatisch Plotfilter und zeigt Plotnotizen + Artenliste

Hinweis zu Google MyMaps:
Falls die MyMaps wegen Google/CORS nicht direkt geladen werden, exportiere sie in Google MyMaps als KML und importiere die KML-Dateien in der App.
